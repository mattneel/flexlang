# DOC001

A doc declaration references a symbol that does not exist.

Docs are checked declarations: `doc name { ... }` and `see name` must
point at real symbols (or bundled modules). Stale references fail the
compile — that is the point.

**Example: docs cannot reference ghosts** — expected to fail with `DOC001` (proven by `flx docs check`):

```flx
doc nonexistent_thing { summary "..." }
fn main() -> I64 = { 0 }
```
